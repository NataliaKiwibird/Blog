HTML Entities

HTML entities are short chunks of code that begin with an ampersand and end with a semicolon. They display reserved characters that would be normally interpreted as HTML code and provide characters that are no available on standard keyboards. You can use an entity name or number to display them on your webpage (not every entity obtains entity name though). Full specification here.
