JS Loops

Loops are a shorten way to go through a piece of code, such as an array or object. They allow us to loop through a data of some kind (scores, grades, list of users etc.).

Piece of code without a loop

Imagine you have an array and you want to log in a consol every element of an array. Without a loop it looks like this:

const meals = ['breakfast', 'brunch', 'lunch', 'dinner', 'supper'];

console.log(meals[0]);
console.log(meals[1]);
console.log(meals[2]);
console.log(meals[3]);
console.log(meals[4]);

Piece of code with a loop

Let's do the same with a use of a for loop. Notice, how much room we saved:

const meals = ['breakfast', 'brunch', 'lunch', 'dinner', 'supper'];

for (let i; i < 5; i++) {
console.log(meals[i])
};

Types of loops:


for, while, do... while


for

To define for, while and do...while loop three elements are needed:
initial expression
condition
increment/decrement expression.
For loop runs a code until the condition returns false.

for (let i = 0; i < 5; i++) {
console.log(i);
};

while

While loop works exactly the same like a for loop, the only difference is a syntax:

let i = 0;
while (i < 5) { console.log(i);
i++
};

do... while

It's an extension of while loop. Useful when you want to loop something that is not likely to be true, but you still want to run it at least once.

let i = 0;
do {
console.log(i);
i++
} while (i < 5);

for of & for in


for of

The for of loop creates a loop iterating over arrays or other iterable objects. The loop is used mostly for objects.

const numbers = [1, 2, 3, 4, 5];

for (let number of numbers) {
console.log(number**2)
};

for in

Used for objects.

let objects = {
'car': 'Audi',
'color': 'red'
};

for (let object in objects) {
consol.log(object)
};

forEach


forEach

It's a method which iterates over an array. forEach runs a function once per every element of array. It uses a function or callback in parenthesis.

let people = ['Harry', 'Ron', 'Hermione'];

people.forEach(person => {
consol.log(person);
});
