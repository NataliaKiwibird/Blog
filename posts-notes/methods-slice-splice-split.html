JS methods:
.slice(), .splice(), .split()

Some of javascript methods have very similar names, that can be really confusing... Despite this, it's worth to understand and remember them, because it makes our programming much easier and it's often subject to interview questions. Here are three similar-sounding and useful methods:

.slice()

.slice() method extracts values from an array and copies them to a new array. It doesn't change the original array. This method can also be used for strings.

example

We have an array with days of week and we want to get weekend days only. We can do it in multiple ways:

syntax

array.slice(start, finish);

bold - required element
normal - optional element

Description of elements:

array - an array that we want to extract values from

start - an index (0 notation) which we start extracting from

finish - an index (0 notation) where we finish the extracting (non-including)

.splice()

.splice() method extracts values from an array and moves them to a new array. The extracted values are removed from the original array! The method can also add elements to the original array.

example

We have an array with a training plan. We want to extract trainings that we've done:

syntax

array.splice(start, numbToDelete, item1, item2, ...itemN);

bold - required element
normal - optional element

Description of elements:

array - an array that we want to extract values from

start - an index (0 notation) which we start extracting from

numbToDelete - number of items we want to delete. If it's 0, nothing will be deleted.

item1 - itemN - items added to the original array.

.split()

.split() method splits (= divides) a string into multiple strings gathered in a new array that is returned. We provide a pattern to divide the string that is placed as a first parameter in the method's call. It's the opposition to .join() array method.

example

We have a confession. We can split this sentence in multiple ways:

syntax

str.split(separator, limit);

bold - required element
normal - optional element

Description of elements:

str - the string that we want to split

separator - The pattern describing where each split should occur. It can be a character or a regular expression. The empty string in parentheses does that each character will be split up.

limit - an integer limiting the number of pieces.
