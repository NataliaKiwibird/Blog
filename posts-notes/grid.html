CSS GRID

CSS GRID turns HTML element into a grid container with rows and columns. You can place children elements wherever you want within a grid. You can also nest grids (nested grids are called subgrids).
According to css-tricks.com, it's "the most powerful layout system available in CSS." It handles two dimensions (columns and rows) in contrast to flexbox, which handles only one selected dimension.

parent element = container
child element = item
one part of grid = cell

CSS GRID I. Properties for grid containers:

display

display: grid;

Must-have of flexbox. Added to a parent element, allows children to be next to each other. Added to a child sets the child in a row.
