CSS flexbox

Flexbox (aka Flexible Boxes) creates page layouts for dynamic User Interface. It is a layout mode that arranges elements in predictable way for different sizes and browsers. Essential for responsive web design.

FLEXBOX I. Properties for flex container = parent of flex items

display

display: flex;

Must-have of flexbox. Added to a parent element, allows children to be next to each other. Added to a child sets the child in a row.

flex-direction

flex-direction: row; (default)
flex-direction: column;
flex-direction: row-reverse;
flex-direction: column-reverse;

Determines the direction of items.

justify-content

justify-content: center;
justify-content: flex-start;
justify-content: flex-end;
justify-content: space-between;
justify-content: space-around;
justify-content: space-evenly;

Sets items on the main axis. Main and cross axis come from flex-direction property. For rows the main axis is and the cross axis is . Inverse for columns.

align-items

align-items: flex-start;
align-items: flex-end;
align-items: center;
align-items: stretch;
align-items: baseline;

Sets items on the cross axis, so for rows and for columns.

flex-wrap

flex-wrap: nowrap; (default)
flex-wrap: wrap;
flex-wrap: wrap-reverse;

Moves items to extra row or column (if they are too wide together).

FLEXBOX II. Properties for child elements

flex-shrink

flex-shrink: 1;
flex-shrink: 2;
flex-shrink: 3; etc.

Allows item to shrink if width of parent is smaller then combined width of all the flex items within it. The higher value the more item shrinks.

flex-grow

flex-grow: 1;
flex-grow: 2;
flex-grow: 3; etc.

The opposite of flex-shrink. Expands items into available space.

flex-basis

flex-basis: Xpx;
flex-basis: Xem;
flex-basis: X%; etc.

Specifies the initial size of item.

flex

flex: 1 0 100px;
flex: auto;
flex: none;
etc.
flex: grow shrink basis;

Shortcut available to set several flex properties at once.

order

order: 1;
order: 2;
order: 3;
order: -1;
etc.

Sets items in a new order, breaks default order.

align-self

align-self: flex-start;
align-self: flex-end;
align-self: center;
align-self: baseline;
align-self: stretch;

Allows to adjust items individually.
